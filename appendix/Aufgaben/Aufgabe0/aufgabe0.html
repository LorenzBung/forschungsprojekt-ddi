Wiederholung HTML

Inhalt dieser Webseite

Diese Webseite ist ein Beispiel dafür, wie ein HTML-Dokument aussehen kann. Sie besteht aus mehreren verschiedenen Elementen, wie

  • Überschriften
  • Textabschnitten
  • Aufzählungslisten
  • Bildern
  • Links

Wofür ist sie da?

Das Übungsblatt enthält mehrere Aufgaben, die zur groben Wiederholung des Themas HTML dienen. Dabei müssen unter Anderem HTML-Elemente verändert werden oder neue Elemente hinzugefügt werden.

Bei Unsicherheiten, was die verschiedenen Tags bedeuten und wie man sie verwendet, kann man auf der Webseite von W3Schools zum Thema HTML viele hilfreiche Informationen und Beispiele finden.
